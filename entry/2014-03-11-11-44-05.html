TwitterのAPIKey新規取得にモバイルの連絡先が必要になっていたので攻略する

最近のことですけど、Twitter連携アプリを作ろうとしたんですね。

で、APIKeyが必要なので、dev.twitter.comにアプリケーション登録を行おうとしたところですね、英語で

「携帯の番号か携帯のメアドを登録せんとあかんで（超訳」

的なことを言われたわけです。

あのお気軽お気楽にAPIKeyを取得できる時代に終焉が訪れた、というところでしょうか。非常に難儀な話です。

とりあえず携帯のメアドを登録してみたが・・・

どうやらDDIポケット時代から愛用しているメールアドレス(********@dk.pdx.ne.jp) では許してくれない様子でした。

実はちょっとした騒ぎになってる模様

あまりにもつらいので、google先生に訊いてみました。

[twitter API 登録 モバイル] [検索]

とかやると、まー出るわ出るわ・・・

俺は！まだ！あきらめない！

Yahoo知恵袋で同じようなことを質問してる人もいる様でしたが、色々調べていると「iPhoneのTwitter公式appでログインすれば通るようになる」とかあったので、これを試してみました。

が、結果はダメ。

最悪のアフォーダンス

次に気になったのがこちらの記事。

twitter.com（devじゃない方）のアカウント設定から、言語設定を英語にすることで、ドキュメントにある通りの、携帯番号の登録が可能になります。

そう。そもそも僕が最初に見た限り、dev.twitter.comでは日本の携帯事業者を選択できない状態でした。

それが何と「言語設定を英語にする」ことで可能になるというアプローチが必要だった模様です。最悪レベルのアフォーダンスなので、本当に修正してほしい。

早速、twitter側の言語設定を英語にしてみたところ、あっけなくAPIKeyの登録に成功。

もしかしたら、DDIポケットのメアドを登録してたり、iPhone上の公式Appでログインしてたりしたからかもしれませんが、電話番号を登録することなく目的を達成できました。

教訓

twitterは言語設定を英語にすべし。

・・・ほんとにこれでいいんだろうかww
